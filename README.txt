موقع مختصر صحيح البخاري — نسخة التصميم الأول مع تحسين النصوص

طريقة التشغيل:
1. افتح ملف index.html مباشرة في المتصفح.
2. أو ارفع المجلد كاملًا على GitHub Pages / Netlify / أي استضافة ثابتة.

ملاحظات:
- التصميم مبني على النسخة الأولى لأنها كانت أقوى بصريًا.
- تم تحسين النصوص فقط لتكون أهدأ، أرقى، وأقرب لأسلوب إنساني طبيعي.
- لا يحتاج الموقع إلى npm أو تثبيت أي مكتبات.
